Java Advanced Math Package

Functions

round(double, int):

Outputs "double" rounded to "int" decimal places

autoRound(double, int):

Outputs "double" rounded to the "int" biggest places

(ie. autoRound(12345.12345, 3) = 12300.0 or autoRound(3.14159265358979, 4) = 3.1416)

random():

Outputs random number from 0 to the integer max value
